Standard Platform League

The SPL robot model format is a UDP message formatted as follows:

id
torso x
torso y
torso z
forward x
forward y
forward z
up x
up y
up z
head_yaw
head_pitch
l_shoulder_pitch
l_shoulder_roll
l_elbow_yaw
l_elbow_roll
r_shoulder_pitch
r_shoulder_roll
r_elbow_yaw
r_elbow_roll
l_hip_yawpitch
l_hip_roll
l_hip_pitch
l_knee_pitch
l_ankle_pitch
l_ankle_roll
r_hip_yawpitch
r_hip_roll
r_hip_pitch
r_knee_pitch
r_ankle_pitch
r_ankle_roll

  • The ID is a single byte, and it is the robot number in [0,4].
  • All other values are floats and encoded as 6-byte floats (same as drawing protocol
  • All joints are in radians
  • Total message size = 1 byte + 31 * 6 bytes = 187 bytes
